# Phantarium 웹뷰어

Phantarium All-in-One의 시장 분석·심층 분석·웹소설 아카이브 데이터를
밖에서 읽는 **토큰 보호 정적 뷰어**.

## 구조 (2026-06 제로베이스 재구축)

- **이 저장소**: 정적 페이지(index.html)만 — 데이터 없음, 공개되어도 안전.
- **데이터**: 오라클 서버 `api.phantarium.co.kr/viewer/data?t=토큰`.
  Phantarium 프로그램의 **[웹뷰어 발행]**이 스냅샷(JSON)을 업로드.
- **접근 제어**: 토큰이 포함된 URL을 가진 사람만 데이터 열람 가능.
  토큰 없는 접속은 안내 화면만 표시. (서버가 403 차단 + CORS 제한)

## 표시 내용

오늘 플랫폼별 랭킹(전일 대비 변동) · 장르 점유율 추이 · 출판사 점유
TOP10/HHI · 랭킹 생존 요약 · 전수 카탈로그 현황 · 웹소설 뉴스.

## 갱신

프로그램에서 [웹뷰어 발행]을 누르면 그 시점의 스냅샷으로 교체된다.
페이지 자체는 push 시 GitHub Actions 가 Pages 로 자동 배포.
